# Ares's Spear

阿瑞斯之矛，JS开发者的利器

## 快速开始

**安装**

```
  npm install ares-spear
```

**使用**

* 全部导入
```javascript
  import * as AresSpear from "ares-spear";
```
* 按需导入,
  需要`babel-plugin-transform-imports`支持

```javascript
  import {WaitFor} from "ares-spear";
```
